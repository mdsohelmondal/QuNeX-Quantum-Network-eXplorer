# QuNeX-Quantum Network eXplorer
**QuNeX** is a Python library for simulating entanglement distribution protocols
in quantum networks.

The library provides tools for:
- Creating five network topologies: Random, Barabási–Albert, Square, Triangular, and Hexagonal.
- Assigning edge parameters to describe network edge attributes.
- Finding multiple edge-disjoint network paths.
- Performing entanglement swapping.
- Implementing multipath entanglement purification (MPEP).
- Computing viable regions and the average viable node fraction for quantum tasks.
